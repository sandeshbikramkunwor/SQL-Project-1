# SQL Analytics Using Window Functions

This project demonstrates real-world business analytics using SQL window functions.
It focuses on customer behavior analysis, sales performance, and employee productivity.

## Key Concepts Covered
- Running totals
- Ranking and Top-N analysis
- Customer segmentation
- Pareto (80/20) analysis
- Window functions (OVER, PARTITION BY, ORDER BY)
- Business-driven SQL problem solving

## Queries Included
1. Running total spend per customer
2. First-time vs repeat customers
3. Customer segmentation based on revenue contribution
4. Pareto analysis (Top 80% revenue customers)
5. Spending spike detection
6. Daily and region-wise running sales
7. Ranking sales per customer
8. Top 2 sales per customer
9. Percentage contribution analysis
10. Employees above department average performance
11. Department performance vs company average

## Tools Used
- SQL Server (T-SQL)
- Window Functions
